# FRCS Exam Preparation

## Exam Hub

The exam prep dashboard gives you an overview of your progress:

* **Overall score** — Animated percentage showing your average.
* **Questions attempted** — Total count.
* **Day streak** — Current consecutive-day practice streak.
* **Best exam score** — Highest score on a simulated exam.
* **Percentile ranking** — See where you stand compared to other members via a score distribution chart.
* **Topic breakdown** — Performance by category (Colorectal, General Surgery, Foundations) with correct/attempted counts.

## Question Bank

Practice with 1,000+ MCQs covering Colorectal, General Surgery, and Foundations topics.

### Practice Modes

{% tabs %}
{% tab title="Untimed Practice" %}
Answer questions with immediate feedback and explanations after each question. Ideal for learning and revision.
{% endtab %}

{% tab title="Exam Simulation" %}
Timed test with no feedback until the end. Simulates real exam conditions.
{% endtab %}
{% endtabs %}

### Features

* **Filter** by topic, difficulty (Easy / Medium / Hard), and subspecialty.
* **Bookmark** difficult questions for later review.
* **Track performance** over time and compare with peers.
* **Flag questions** if you spot an issue (incorrect answer, outdated info, unclear wording, wrong topic, duplicate).
