# Education

## Question Bank

Manage the exam revision question bank from the Questions page. Questions are used by members for self-assessment and exam preparation.

### Question Types

| Type               | Description                                        |
| ------------------ | -------------------------------------------------- |
| MCQ                | Multiple choice — one or more correct answers      |
| Single Best Answer | Multiple options but only one correct answer        |
| Extended Matching   | Match items from two lists                         |

### Creating a Question

1. Click **Add Question**.
2. Enter the **question text**.
3. Optionally upload a **question image** (e.g. a clinical photo or diagram).
4. Set the **difficulty level**: Easy, Medium, or Hard.
5. Select a **topic** from three main categories:
   * **Colorectal** — 11 subtopics (e.g. Colorectal Cancer, IBD, Diverticular Disease, Pelvic Floor).
   * **General Surgery** — 9 subtopics (e.g. Acute Abdomen, Hernia, Upper GI).
   * **Foundations** — 8 subtopics (e.g. Anatomy, Physiology, Pathology, Pharmacology).
6. Add **answer options** (up to 6):
   * Type each option text.
   * Click the green highlight button on the correct answer(s).
   * Reorder options as needed.
7. Write an **explanation** for the correct answer — this is shown to members after they answer.
8. Set **status** to Draft or Published.

### Flag System

Members can flag questions they believe have issues. Flags appear in the admin panel with a reason:

| Flag Reason      | Description                                        |
| ---------------- | -------------------------------------------------- |
| Incorrect answer | The marked correct answer may be wrong             |
| Outdated info    | The question references outdated guidelines        |
| Unclear wording  | The question or options are ambiguous              |
| Wrong topic      | The question is categorised under the wrong topic  |
| Duplicate        | The question is a repeat                           |
| Other            | Any other issue                                    |

> **Warning:** Review flagged questions regularly and either update the question or dismiss the flag. Unresolved flags affect member trust in the question bank.

### Search and Filter

* Search questions by text.
* Filter by difficulty level.
* Filter by topic/category.

## Fellowships

Manage fellowship programme listings from the Fellowships page.

### Creating a Fellowship

1. Click **Add Fellowship**.
2. Fill in the sections below.

### Basic Information

* **Title** — Fellowship name (e.g. "Pelvic Floor Fellowship, St Mark's Hospital").
* **Slug** — Auto-generated from the title.
* **Description** — Full programme description.
* **Featured Image** — Upload or select an image.
* **Status** — Draft, Published, or Archived.

### Programme Details

* **Duration** — 6 months or 12 months.
* **Subspecialties** — Tag with relevant surgical subspecialties.
* **Accreditation** — Select accrediting bodies: ACPGBI, ESCP, RCS England, RCS Edinburgh, RCS Ireland, RCPS Glasgow, or Other.

### Application and Eligibility

* **Eligibility Criteria** — Free-text description of requirements.
* **Training Level Requirements** — Checkboxes for eligible training stages.
* **Application Deadline** — Cut-off date.
* **Custom Application Questions** — Add bespoke questions with type selectors and required/optional toggles.

### Supervisor / Lead

* **Database Mode** — Search the [Faculty database](people.md#faculty) and select an existing profile. Use this when the supervisor is already in your system.
* **Manual Mode** — Type in names directly. Use this for external supervisors who are not in the Faculty database.

Toggle between modes using the DB/Manual switch.

### Financial Information

* **Salary Range** — Minimum and maximum salary.
* **On-call Pattern** — Select from: None, 1 in 2, 1 in 2-3, 1 in 3, 1 in 4, 1 in 5, 1 in 6, 1 in 7, 1 in 8+.

### Locations

Select one or more [institutions](organisation.md#institutions) from the database. Each institution shows its name, city, and country.
