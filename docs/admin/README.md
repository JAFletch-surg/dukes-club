## Quick Reference

### Status Workflow
All content types use: **Draft** → **Published** → **Archived**

### Roles

| Role | Access |
|------|--------|
| Pending | No access — awaiting approval |
| Trainee | Standard member features |
| Member | Standard member features |
| Editor | Can manage content |
| Admin | Full admin panel access |
| Super Admin | Full access including member management |

### Key Workflows

- **New member sign-up** → Review in [Members](people.md#members) → Approve/Reject → Email sent
- **New event** → Create in [Events](events.md) → Assign faculty → Open applications → Manage applicants → Collect feedback → Issue certificates
- **Video library** → Sync from Vimeo in [Videos](content.md#videos) → Tag and publish
- **Question bank** → Add questions in [Education](education.md#question-bank) → Review flags from members
